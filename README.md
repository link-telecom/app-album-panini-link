# App Album Panini - LINK

PWA para gestionar el álbum Panini FIFA World Cup 2026 con amigos en tiempo real.

## Stack

- Next.js 15 + React 19 (App Router)
- TypeScript
- Tailwind CSS v4
- Firebase (Auth + Firestore)
- PWA instalable

## Desarrollo local

```bash
npm install
npm run dev
```

Abre http://localhost:3000

## Variables de entorno

Copia `.env.example` a `.env.local` y rellena con tus credenciales de Firebase.
